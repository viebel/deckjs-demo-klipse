Interactive slides example

The slides are built with RevealJS, a slide deck generator using Asciidoctor, a powerful toolchain for publishing.

The code snippets are evaluated by Klipse, an interactive client-side code evaluator.

A template for this type of presentations is available in asciidoctor-revealjs-klipse github repository.

RevealJS navigation help can be accessed by pressing "?".

Press the "right arrow" key to move to the next slide.
